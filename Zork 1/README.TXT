Playing the Game
-----------------------------------------------------------------------------------------------------
Windows 95
-----------------------------------------------------------------------------------------------------
 In order to play the game through Windows 95, complete the following tasks:

Press the "Start" button on the Taskbar
Select the "Run" option
Press the "Browse" button, and locate the Zork 3 files (by default, they will be 
   extracted to the Zork3 folder).
Select the "zork3.bat" file, and click "Open"
Press "OK" from the "Run" menu.

Windows 3.1
-----------------------------------------------------------------------------------------------------
In order to play the game through Windows 3.1, do the following:
Run File Manager
Open the "Zork3" folder 
Double-click on the "Zork3.bat" file

DOS
-----------------------------------------------------------------------------------------------------
To play the game under DOS,

from the C> prompt, type "cd\zork3"
type "zork3"

Gameplay notes
-----------------------------------------------------------------------------------------------------
The game is a text adventure and recognizes different words that you type in to 
the computer.  The following is a list of some (but not all) of the verbs that 
you can use:

look		take
examine		kill
talk		open
close		enter
exit		push
pull		lift
move		tie

You can also use compass directions to indicate that you want to move in that 
direction.  For example, you can type:

"go north"

or you can type

"north"

or just

"n"

Typical directions are N,S,E,W,NE,NW,SE,SW,Up,Down

To save a game, type "save" and the name of the game you want to save.

To load a game, type "restore" and the name of the game you want to load.

Have fun, and look for Zork Grand Inquisitor in November!
